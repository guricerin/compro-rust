# compro-rust

Workspace for competitive programing with Rust.

## Requirement

* bash
* kmyk/online-judge-tools (https://github.com/kmyk/online-judge-tools)

## Usage

* setup or change problems

```bash
chmod a+x ready.sh
./ready.sh
```

* build and test

```bash
chmod a+x test.sh
./test.sh (a ~ d)
```
